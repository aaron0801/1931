解鎖檔案

FILE 01｜1931：封存起點 1931 是左鎮化石記憶被打開的年份，也是本作封存編號的來源。
1931 代表左鎮化石故事的起點。作品把這個年份轉化成封存系統的核心密碼，讓玩家在解謎後理解：檔案不是單純資料，而是地方記憶被重新打開的入口。
FILE 02｜化石形成條件 化石需要掩埋、時間、壓力與礦物質滲入，才可能保存生命痕跡。
遺骸不會直接變成化石。它需要被快速掩埋，經過長時間壓力與礦物質滲入，才有機會把原本的生命痕跡轉化成地層中的紀錄。
FILE 03｜真正任務 玩家復原的不是普通雞骨，而是把左鎮化石知識轉化成互動學習體驗。
黑暗研究室只是包裝。真正被保存的是左鎮化石的地方知識、歷史記憶與學習經驗。玩家完成的不是恐怖任務，而是一場化石教育轉譯。
